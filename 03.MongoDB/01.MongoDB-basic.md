## MongoDB Basic:

MongoDB is developed by **MongoDB Inc.** and is available in multiple editions depending on deployment type and support needs.

### MongoDB Editions Overview:

#### 1. MongoDB Community Edition (Free): 

Open-source version (SSPL license)
- Free to use
- Replica Set (Replication)
- Sharding
- WiredTiger storage engine
- Indexing & Aggregation framework
- Basic security (authentication, RBAC, TLS)
- 📌 Best for: Development, labs, startups, self-managed production


#### 2. MongoDB Enterprise Advanced (Paid):

Commercial subscription from MongoDB Inc. Includes everything in Community plus:
- LDAP / Kerberos integration
- Advanced auditing
- In-memory storage engine
- Encryption at Rest
- Ops Manager (on-prem management)
- Enterprise support (24/7)
- 📌 Best for: Banking, telecom, regulated industries.


#### 3. MongoDB Atlas (Fully Managed Cloud):

Managed DBaaS by MongoDB Inc. MongoDB Atlas Features:
- Fully managed clusters
- Auto-scaling
- Automated backup
- Global clusters
- Built-in monitoring
- Multi-cloud (AWS, Azure, GCP)
- 📌 Best for: Production workloads without managing infrastructure.
